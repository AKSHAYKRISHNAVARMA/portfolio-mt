# Akshay Krishna Varma Buddharaju — Portfolio

**Live site:** https://akshayb.vercel.app

Personal portfolio for AI/ML Engineer, Data Scientist, and Generative AI roles.
Built with Next.js, React, Tailwind CSS, and Framer Motion. Deployed on Vercel.

## Sections

- **Hero** — introduction and current focus across AI engineering and data science
- **Impact** — measurable results from internships and projects (RAG latency, retrieval efficiency, model accuracy)
- **Featured Work** — Video RAG system (The Home Depot CoreAI), computer vision, NLP, and GNN projects
- **Experience** — roles across ML engineering, data science, and software engineering
- **AI & Data Science Toolkit** — tabbed skills across ML/Data Science, Generative AI & NLP, Computer Vision & Deep Learning, and Cloud & MLOps
- **Education** — M.S. in Computer Science (Data Science concentration), Kennesaw State University
- **Research & Publications** — "Semantic Segmentation of Remote Sensing Images of Urban Areas using Deep Learning Methods," International Journal of Research Publication and Reviews (2024)
